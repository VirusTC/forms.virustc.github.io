# FDA Regulatory Disclaimer: General Reference & Administrative Module

## 1. Non-Device Clinical Decision Support (CDS)
This module functions strictly as a general reference tool, administrative aid, or Non-Device Clinical Decision Support (CDS) application under FDA guidelines. It does not perform real-time patient monitoring, image analysis, or predictive diagnostic calculations.

> **NOTICE:** This software is intended solely to display, organize, store, or print existing medical information and literature. It does not independently diagnose, cure, mitigate, prevent, or treat any disease, condition, or health anomaly.

## 2. Information Currency & Accuracy
The reference materials, clinical guidelines, and documentation displayed within this tool are compiled from public and peer-reviewed medical sources. While we strive to keep this information updated, medical knowledge changes rapidly. Users are responsible for confirming that the referenced guidelines align with current medical standards and institutional protocols.

## 3. Exclusion of Emergency / Real-Time Use
This software is strictly prohibited from use in active patient monitoring, emergency triaging, or time-critical clinical environments. It is not designed to trigger urgent alerts, notifications, or life-saving interventions.
