# FDA Regulatory Disclaimer: Real-Time / SaMD Software Module

## 1. Professional Clinical Judgment
The real-time data, alerts, and analytical outputs provided by this software are intended strictly to support qualified healthcare professionals (HCPs). This software does not make automated diagnostic or treatment decisions. 

> **CRITICAL NOTICE:** This software is not a substitute for the independent professional judgment, diagnosis, or treatment planning of a licensed medical practitioner. Clinicians must independently verify all critical alerts, recommendations, and patient data before initiating any medical intervention.

## 2. Intended Use & High-Acuity Limitations
This module is designed for real-time data processing and active patient monitoring. However, it must not be used as the sole basis for life-support decisions or immediate, high-acuity interventions without secondary clinical verification. System latency, network connectivity issues, or data transmission drops may delay updates. Clinicians must rely on primary bedside monitoring systems and direct patient assessment in life-threatening situations.

## 3. Data Integrity & Input Dependencies
The accuracy of the real-time outputs generated by this system depends entirely on the completeness and accuracy of the data ingested from connected hospital systems (e.g., EHR, bedside monitors). This software is not responsible for misinterpretations or false alerts caused by corrupted, delayed, or missing incoming data streams.
